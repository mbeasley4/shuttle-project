## Project Overview

This theme was developed to:

- Replace a previously purchased third-party Shopify theme  
- Deliver a design that better represents The Shortening Shuttle brand  
- Improve product page layout and content structure  
- Simplify ongoing content management for a small internal team  
- Clean up legacy markup, styling, and theme bloat  
- Provide a scalable foundation for future store enhancements  

---

## Tech Stack

- Shopify Online Store 2.0  
- Liquid templating  
- JSON templates  
- CSS / JavaScript  
- Shopify CLI for local development  
- Git for version control
